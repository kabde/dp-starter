# DP Starter — WordPress Theme

A versatile, premium WordPress starter theme for digital product creators, coaches, and consultants.

## Features (Free)

- Full design system (colors, typography, spacing) configurable from admin
- 113 Google Fonts
- Light & dark mode support
- Configurable home page (9 sections with on/off toggles)
- Responsive header, footer, sidebar
- Page hero custom fields
- Blog with reading time, categories, related posts
- 404, search, archive templates
- Custom CSS support

## Premium Features (License Required)

Unlock with a license key in **Appearance → DP Starter Settings → License**:

- Custom Post Types (Books, Tools, Advertorials, Offer Links)
- WooCommerce checkout optimization
- Lead capture with FluentCRM/MailPoet auto-tagging
- UTM & Facebook Pixel tracking
- Offer page builder with product cards
- Article CTA banners (inline + sticky)
- Maintenance mode with countdown
- Custom code injection (head & footer)
- Admin comment creator

## Installation

1. Download the latest release ZIP
2. In WordPress: **Appearance → Themes → Add New → Upload Theme**
3. Activate the theme
4. Go to **Appearance → DP Starter Settings** to configure
